# 📚 Student Marks Predictor

## 📌 Overview

This project predicts student marks based on study hours using Linear Regression.

---

## 🚀 Technologies Used

- Python
- Pandas
- Scikit-learn

---

## 📂 Dataset

The dataset contains:

- Hours Studied
- Marks Obtained

---

## ▶️ How to Run

```bash
pip install pandas scikit-learn
python student_marks_predictor.py
```

---

## 📈 Example Prediction

Hours = 5

Predicted Marks = 60
